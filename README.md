#Unity3D-prpg
Procedural 2D Roleplaying Game in Unity 3D

##Requirements:
- 2D Toolkit from the Unity 3D asset store

##Goal:
The goal of this game is to create a Dragon Warrior 3 style 2D RPG procedurally.
